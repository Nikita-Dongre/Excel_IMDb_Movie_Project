# End-to-End Excel IMDB Movie Project

A complete Excel-based analytics project built using IMDB movie data. This project demonstrates the full workflow of data analysis—from raw data cleaning to building an interactive dashboard—using Excel tools such as Pivot Tables, Pivot Charts, and calculated fields.

---

## Project Objectives

- Clean and prepare raw IMDB data  
- Transform and structure data for analysis  
- Build Pivot Tables and Pivot Charts  
- Create an interactive dashboard  
- Develop insights and analytical storytelling  
- Compute custom metrics using calculated fields

## How to Use This Project

1. Start with **8_Raw_Data.xlsx** to view the original dataset.  
2. Explore **7_Clean, Structured data.xlsx** to review the cleaned dataset.  
3. Open **4_Pivot_Tables.xlsx** and **3_Pivot_Charts.xlsx** to see how insights were built.  
4. Review custom metrics in **5_Calculated_Fields _ Metrics.xlsx**.  
5. Open **2_Interactive_dashboard.xlsx** to interact with the final dashboard.
